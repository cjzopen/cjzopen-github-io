CSS Variables

2017-06-29


TL;DR

CSS3 其實有原生的變數系統：前面加上 -- 。


現況

SASS 當道的情況下，其實沒什麼人會注意到 CSS3 的變數系統。除了 SASS 之外，其實還有瀏覽器相容性的問題，不過這個問題在今年也幾乎能忽略了：Edge 15 之後的版本也能使用變數功能了，不過需要注意 Edge 在 animation 或是 pseudo elements 使用的話可能會出現 bug 。

範例

//設定變數
:root {
  --main-color: #693905;
  --main-bg: #af946e;
}

//或者變數寫在元素裡面
.element {
  --main-bg: #af946e;
}


.element {
  background-color: var(--main-bg);
}

世界遠離 SASS 又近了一步(?
