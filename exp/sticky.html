position: sticky;

2017-06-13


TL;DR

css 語法 position: sticky; 從原本的行動裝置再次(? 回歸各大瀏覽器，除了用在傳統的上下方 menu bar 之外，還可以用在與 scroll bar 之間的互動。


現況

在經歷了只有行動裝置裡的 safari 有支援這項語法的黑暗期之後，2016 年 google 正試宣佈新版的 chrome 將會再次支援 sticky 這項 css 語法。其實不只是 chrme ，各家瀏覽器也相繼支援，連從來都不鳥這項語法的 IE ( Edge ) 也將在 Edge 16 開始支援 ( 6 月時最新的版本為 15 )。但請注意， table 並不適用這項屬性。

功能

這項語法的功能很簡單：在螢幕範圍內時是 position: static; ，超過範圍時變成 position: fixed; ，而且其它物件不會因此跑版。運用的地方像是傳統 fixed 在上方的 menu bar 、Facebook 和 bilibili 播放中的影片，它把原本寫 js 判斷 scrollTop 的時間都省下來了。 


position: -webkit-sticky;
position: sticky;
z-index: 1030;

/* 一定要給定位置它才會理你 */
top: 0;

簡單來說，就是讓開發者與 scroll 事件之間有更多靈活的運用空間。
