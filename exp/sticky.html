position: sticky;

2017-06-13

在經歷了只有行動裝置裡的 safari 有支援這個語法黑暗期之後，2016 年 google 正試宣佈新版的 chrome 將會再次支援 sticky 這項 css 語法。其實不只是 chrme ，各家瀏覽器也相繼支援，連從來都不鳥這項語法的 IE ( Edge ) 也將在 Edge 16 開始支援。

功能

這項語法的功能很簡單：在螢幕範圍內時是 position: static; ，超過範圍時變成 position: fixed; ，而且其它物件不會因此跑版，把原本寫 js 的判斷 scrollTop 的時間都省下來了。
最常應用且最簡單的地方應該就是傳統 fixed 在上方的 menu bar 了。
