用 box-shadow 產生底線

2017-08-07


inset 與 hover 之後 inset 透明

main article span{
    box-shadow:0 -1px 0 #2858aa inset;
    transition:box-shadow .2s ease-in-out;
}
main article span:hover{
    box-shadow:0 -1px 0 rgba(0,0,0,0) inset,
               0 3px 0 #2858aa;
    transition:box-shadow .2s ease-in-out;
}
